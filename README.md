# PROYECTO-PROGRAMACION-2

# Gestion de cultivos

## Breve descripción del problema que resuelve
Este proyecto proporciona una solución digital para la planificación, seguimiento y optimización de cultivos. Permite a los agricultores registrar actividades agrícolas, gestionar recursos y mejorar la productividad mediante un control estructurado de sus cultivos.

## Lista Inicial de Módulos del Sistema

### 1.	Registro y Seguimiento de Cultivos
	-	Permite registrar información sobre cultivos, fechas de siembra, cosecha y tratamientos aplicados.
	-	Historial de cada cultivo para un mejor control y planificación futura.
 ### 2.	Gestión de Recursos Agrícolas
	-	Control de insumos como fertilizantes, pesticidas y semillas.
	-	Registro del uso de maquinaria y herramientas para optimizar su mantenimiento.
 ### 3.	Planificación y Reportes
	-	Generación de cronogramas de actividades agrícolas.
	-	Informes sobre rendimiento de cultivos y costos asociados.
